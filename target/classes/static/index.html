Sobre o Projeto

Inteligencia em Movimento

O projeto utiliza sensores de contagem bidirecional para acompanhar ocupacao em tempo real sem capturar imagem, audio ou biometria.

S

Sensor VS350-B15M

Contagem precisa de entradas e saidas com foco em privacidade.

D

Inteligencia de Dados

Indicadores por hora, dia e setor para apoiar decisoes operacionais.

L

Conformidade LGPD

A solucao trabalha apenas com contagens anonimas de fluxo.

Acesso

Area do Usuario

Login e cadastro usam diretamente o backend Java. As rotas sensiveis tem limite de requisicoes por IP para reduzir abuso automatizado.
